# Drive Cloner

A compiled GTK3-based disk cloning and image creation utility for Linux systems.

## Features

- **Disk Cloning**: Clone entire disks or partitions
- **IMG Creation**: Create disk images from drives
- **Progress Tracking**: Real-time progress display with speed and time estimates
- **Root Mode Support**: Automatically detects and handles root privileges
- **Modern GTK3 Interface**: Clean and intuitive user interface

## Usage

1. Download the `clone.py`  file
2. Run with root privileges: `sudo python3 clone.py`

## Important Notes

- **⚠️ WARNING**: This tool can permanently erase data. Use with caution!
- Always backup important data before using disk cloning tools
- The application requires root privileges for disk operations
- Make sure to select the correct source and destination drives

## System Requirements

- Linux system with GTK3 libraries
- Root privileges for disk operations
